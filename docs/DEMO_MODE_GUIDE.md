# Demo Mode Deprecated

Demo Mode has been replaced by Feature Lab.

Use:

```text
/#/lab
```

Legacy demo URLs redirect to Feature Lab:

- `/#/demo`
- `/#/demo/story`
- `/#/demo/staff`
- `/#/demo/system`
- `/#/demo/attendance`
- `/#/demo/emergency`

Do not add new Demo Mode pages. Future preview/prototype ideas should be documented in `docs/FEATURE_LAB_GUIDE.md` and implemented under `/lab` only.

Feature Lab must remain mock-data-only and must not include Parent Orientation, Document Center, document generation, public applicant demos, or production-like application flows.
